NNYFrameworks
=============

No Name Yet Frameworks

Quick Start
-

node.js<br>
http://nodejs.org/

webpack module bundler<br>
http://webpack.github.io/

rogerwang/node-webkit<br>
https://github.com/rogerwang/node-webkit

Install:
<pre>$ cd /path/to/</pre>
<pre>$ npm install</pre>

Bundle:
<pre>$ cd apps/myApps/emptyExample</pre> 
<pre>$ webpack --watch</pre>

Run:
<pre>$ nodewebkit</pre>
